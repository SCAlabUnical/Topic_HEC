per ottenere i risultati riportati nell'elaborato di tesi settare gli hyper-paramaters del file main.py cosi come riportato nel file configuration.txt
per l'esecuzione eseguire "python3 main.py"
in caso di errore : "OSError: /home/USER/.local/lib/python3.9/site-packages/torch/lib/libgomp-d22c30c5.so.1: cannot allocate memory in static TLS block" eseguire :"export LD_PRELOAD=/home/USER/.local/lib/python3.9/site-packages/torch/lib/libgomp-d22c30c5.so.1"
